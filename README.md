# Chandralekha E-Commerce Store

## 📌 Project Description

Chandralekha E-Commerce Store is a responsive online shopping website developed using HTML, CSS, and JavaScript. This project was created as part of ApexPlanet Web Development Internship Task 5 (Final Project and Optimization).

The website provides a modern shopping experience with product search, category filtering, price sorting, shopping cart functionality, wishlist feature, product detail view, dark mode, and mobile responsive design.

## 🚀 Features

- 24 Product Items Display
- Product Search Bar
- Category Filter
- Price Sorting (Low to High / High to Low)
- Add to Cart Functionality
- Buy Now Button
- Wishlist Feature
- Product Details Popup
- Cart Item Counter
- Total Cart Price Calculation
- Dark Mode Toggle
- Lazy Loading Images
- Mobile Responsive Design
- Cross-Browser Compatibility
- Clean and User-Friendly Interface

## 🛠 Technologies Used

- HTML5

- JavaScript
